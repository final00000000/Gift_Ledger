## 📊 数据统计
- **收支概览**：一目了然的收支对比卡片
- **实时余额**：动态计算收支差额
- **可视化图表**：直观展示收支趋势

## 📝 记录管理
- **快速记账**：简洁的记账界面，支持自定义金额输入
- **智能建议**：根据历史记录提供联系人建议
- **灵活分类**：支持婚礼、满月、乔迁、生日、丧事、过年等多种事由
- **关系标签**：朋友、同事、亲戚、同学等关系分类
- **农历日期**：支持农历日历选择，方便记录传统节日

## 📚 活动簿与批量录入
- **活动簿**：按活动管理礼金记录（婚礼、满月酒等）
- **批量录入**：在活动簿内快速添加多条记录
- **活动统计**：按活动维度查看收支与笔数

## 👥 人际管理
- **联系人管理**：自动维护礼金往来的联系人列表
- **往来追踪**：清晰记录每个人的收礼和送礼金额
- **还礼状态**：待还礼、已还礼清晰标记
- **进度可视化**：直观显示还礼进度条

## 🔐 安全与隐私
- **安全锁**：PIN 码保护敏感信息
- **隐私显示**：金额隐藏模式，保护私密场景
- **解锁体验**：无锁模式不弹 PIN，流程更顺畅

## 🔔 提醒与话术
- **每月提醒**：自动推送待还礼 Top 3
- **话术模板**：自定义提醒消息模板
- **模糊金额**：提醒话术支持“千把块”等模糊描述

## 📦 数据与备份
- **导出备份**：支持 Excel / JSON
- **导入恢复**：支持从备份恢复数据
- **严格验证**：防止重复导入，保障数据完整性

## 🎨 界面与体验
- **现代 UI**：Material Design 3 设计语言
- **流畅动画**：精心设计的页面过渡与交互动画
- **响应式布局**：适配不同屏幕尺寸

## 🧩 多端支持
- **Android / Windows / Web / iOS**：多平台一致体验
